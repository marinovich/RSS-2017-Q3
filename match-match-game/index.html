Skirt Card
Game Difficulty
  • Easy (3x2)
  • Medium (4x3)
  • Hard (6x4)

How to play:

Memory is a counter game where the object is to find the pairs.
When the game begins, all pictures are hidden.

To play:

  1. Select two cards to match the pictures.
  2. If you match the pictures you can go again.
  3. If they don't match it is the computer turn them.
  4. The player that finds all pairs wins!
  5. Have Fun!

Congratulations!
